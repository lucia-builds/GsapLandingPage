# Itzfizz Landing Page (pracapp)

A high-end, immersive landing page built with **React**, **Vite**, **GSAP**, and **Three.js**.

## 🚀 Features

- **Custom Interactive Loader**: A polished progress-based entry animation.
- **GSAP & Framer Motion Animations**: Smooth, high-performance scroll-triggered animations.
- **Lenis Smooth Scroll**: Enhanced scrolling experience for a premium feel.
- **Three.js Starry Background**: An interactive, performance-optimized particle background.
- **Responsive Components**:
  - **AnimatedHeader**: Dynamic text reveals on scroll.
  - **HorizontalScroll**: Immersive horizontal content navigation.
  - **CardStack**: Interactive layering of content.
  - **ImpactMetrics**: Engaging data visualization.
- **Tailwind CSS 4**: Modern utility-first styling.

## 🛠️ Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite
- **Animations**: GSAP, @gsap/react, Framer Motion
- **3D Graphics**: Three.js, @react-three/fiber, @react-three/drei
- **Smooth Scroll**: Lenis
- **Styling**: Tailwind CSS 4
- **State/UI**: Leva (GUI controls)

## 📦 Getting Started

### Installation

```bash
cd pracapp
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

### Build

Build the project for production:

```bash
npm run build
```

## 📂 Project Structure

- `src/components/`: Modular UI components (Navbar, Footer, Projects, etc.).
- `src/App.jsx`: Main entry point with custom loader and layout logic.
- `src/index.css`: Global styles and Tailwind configuration.
